# Placeholder Entity Atlas

`placeholder_entities_atlas.png` uses a 16 px base grid, but entity regions can span multiple cells.

- `(0,0) 16x16`: worker bot
- `(16,0) 16x16`: hostile scout
- `(32,0) 16x16`: support drone
- `(48,0) 16x16`: loose utility marker
- `(0,16) 32x32`: medium bot
- `(32,16) 32x32`: medium enemy
- `(64,16) 48x32`: large enemy / machine
- `(0,48) 32x16`: turret / floor fixture
