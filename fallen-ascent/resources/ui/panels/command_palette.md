# Command Palette (`command_palette.png`)

Bottom-center command palette: tabs (Orders/Zones/Rooms/Structures/Objects)
plus their command icon grid. Built in `scripts/ui/ColonyHud.gd`
(`CommandPalette`, around line 282).

- Anchored: bottom-center.
- Width ~`PALETTE_WIDTH` (580 px), height ~`PALETTE_HEIGHT` (200 px).
- Tone: heavy industrial frame; the icons themselves come from
  `placeholder_ui_atlas.png` and need to read against this background.
