# Worker Atlas

`worker_atlas.png` uses 32 px cells.

Columns:

- `(0,0)`: worker facing south
- `(32,0)`: worker facing south east
- `(64,0)`: worker facing east
- `(0,32)`: worker facing north east
- `(32,32)`: worker facing north 
- `(64,32)`: worker facing north west
- `(0,64)`: worker facing west
- `(32,64)`: worker facing south west


Worker facing updates from movement vector and defaults south while idle.
