# Structures Atlas

`structures_atlas.png` uses 32 px cells. Structures do not have direction variants.

Columns:

- `(0,0)`: wall
- `(32,0)`: door
- `(64,0)`: light
- `(96,0)`: extractor
- `(128,0)`: sensor
- `(160,0)`: charge pad
- `(192,0)`: fabricator
- `(224,0)`: dock
- `(256,0)`: repair bench
- `(288,0)`: assembly press
- `(320,0)`: mechanic dock
- `(352,0)`: calibration shrine
- `(384,0)`: meditation pad
- `(416,0)`: sentience cradle

Column order matches `BuildBlueprint.Id`.
